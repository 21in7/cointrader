# 크래시 포착(crisis-alpha)이 ETH 고유인가 크립토 일반 성질인가 — 8자산 검증

## 목표·가설
ETH walk-forward에서 본 "추세추종 크래시 포착(crisis-alpha/convexity)"이 ETH 고유인지
크립토 전반의 일반 성질인지 falsification-first로 검증. 가설 = **"일반 성질이다."**
ETH에서만 → ETH-specific(아침 아티팩트 의심 일부 옳음). 여러 자산·에피소드서 일관 → 일반.

## 방법
- ETH와 동일 anchored walk-forward(train=2yr/test=6mo/step=6mo, 고정 L=2w + 적응형 L,
  TSMOM long/short)를 8자산(BTC ETH XRP SOL AVAX LINK DOGE TRX) spot 일봉에 적용.
- 크래시 레짐 분해 — (i) BTC trailing-peak DD > 30%, (ii) 명명 에피소드 5개(2018베어,
  2021-05, LUNA, FTX, 2022베어). stitched OOS를 크래시 vs 평시 / 에피소드별로 분해.
- 상관 보정: 크립토 크래시는 상관↑ → fold 아니라 *구별 에피소드* 단위 집계.
- 모듈: `src/momentum/crisis_alpha.py` (walkforward/precheck 재사용).

## 사전등록 "일반 성질" 기준 (3개 AND)
1. 크래시+ & 집중(crash>normal) 자산 ≥ 5/8
2. 자산 전반 +포착 에피소드(존재자산 과반 +) ≥ 3
3. 평시 드래그 부호 일관(음) ≥ 5/8

## 결과 — ❌ 일반 성질 REJECTED (적대적 검증 통과: REJECTED-with-caveats)

| 게이트 | 결과 | |
|---|---|---|
| 1. 크래시집중 ≥5/8 | **4/8** | BTC·XRP·SOL·AVAX |
| 2. 에피소드 ≥3 | **2/5** | 2021-05(4/6), LUNA(5/6)만. FTX **1/8**, 2022베어 2/8 |
| 3. 평시드래그 ≥5/8 | **2/8** | 대부분 평시에도 +(드래그 아님) |

**임계 강건성**: DD 임계 0.20/0.25/0.30/0.40 전부에서 게이트 conjunction = False
(게이트2는 임계무관 고정에피소드, 게이트3은 최대 3/8). 판정 안 흔들림.

### 핵심 발견 — 크래시 포착은 "크래시 형태(morphology)"에 의존
에피소드×자산 매트릭스가 결정적:
- **LUNA-2022-05(완만한 하락): 5/6 자산 +포착** — 추세추종이 따라잡을 시간이 있는 grind.
- **FTX-2022-11(급격한 충격): 8자산 중 7개 손실(1/8만 +)** — 추세 전환 전 급반전 = 휩쏘.
broad-based(FTX 7/8 음, 단일자산 아티팩트 아님). → "추세추종이 크래시를 잡는다"는
**모든 크래시가 아니라 "잡을 시간을 주는 완만한 크래시"에만** 성립.

### ETH 재조명 — "크래시 포착"이라 부르기도 어려움
ETH crashPnL +232% < **normPnL +264%** (크래시는 ETH 총 PnL의 47%). 즉 ETH는 평시에
*더* 벌었다 → ETH의 엣지는 크래시 컨벡시티가 아니라 **전반적 추세 + 이상치 2개 에피소드**
(LUNA +101, 2022베어 +119 = 총 496% 중 ~44%). 아침의 단독 walk-forward 통과는 진짜였으나
**ETH-specific**였고, "크래시 포착"으로의 일반화는 데이터가 지지하지 않음.

### SOL/AVAX "크래시집중"의 정체 = slow-grind, shock 포착 아님
SOL·AVAX는 게이트1에 카운트되지만(crash_pnl>0), **그들이 커버한 명명 에피소드는 전부 손실**
(SOL FTX −53%·2022베어 −61%; AVAX FTX −48%·2022베어 −58%). 양(+) crash_pnl은 전부 명명
에피소드 밖 **slow-grind 블록**에서 옴. 즉 "DD>30%(=ATH 아래) 장기 침체기에 숏 유지"이지
이산적 크래시 *포착*이 아니다.

### 다자산 EW 추세 포트폴리오 [★진단용, 게이트 아님]
- 정정: 초안의 inner-join 버그(2022-09~2025-12로 잘려 LUNA/2021 누락)를 **적대적 검증이
  발견** → outer-join(날짜별 가용자산 평균)으로 수정. 헤드라인 반전:
  **EW Sharpe 0.58 > buy&hold 0.45, crashPnL +148%**(버그판 0.22<0.51, −7%는 폐기).
- 단, 이 EW는 per-asset 승자(ETH/AVAX/SOL)+높은 상관에 지배됨 → **"일반 성질"의 증거 아님**.
  판정(게이트 1·2·3)은 EW를 전혀 쓰지 않으며 EW 정정과 무관하게 REJECTED 유지.

## 결론
**크래시 포착은 크립토의 일반 성질이 아니다.** (1) 자산 전반으로 균일하지 않고(4/8, 대부분
저조), (2) 크래시 형태 의존적이며(LUNA 포착 vs FTX 휩쏘), (3) ETH의 "엣지"는 크래시
컨벡시티가 아니라 추세+이상치 에피소드였다. 다자산 EW가 buy&hold를 이기긴 하나 소수 승자와
상관에 지배된 진단 지표일 뿐, 예측 알파도 확정된 컨벡시티도 아니다.

→ 이 라인(크립토 directional 추세/크래시)에서 **이식 가능 알파는 확인되지 않음.** 만약
crisis-convexity를 진지하게 보려면 **상관 낮은 비-크립토 자산군(주식·채권·상품 — CTA crisis-alpha
문헌이 입증한 곳)** 이 필요 = 새 데이터·후속 영역. 현 8-크립토 유니버스로는 결론 불가.

## 방법론 메모 (적대적 검증의 가치)
이 결과는 적대적 검증 workflow(3 렌즈: 구현/통계/프레이밍 + 종합)를 거쳤다. 그 결과:
- **EW inner-join 버그 발견** → 잘못된 주장("EW 열위, 크래시 −7%") 커밋 직전 차단·정정.
- 핵심 판정(REJECTED)은 모든 임계에서 강건함을 독립 재현(per-asset 엔진 causal 검증:
  pos shift, crash+normal==total 1e-16, NaN 0).
- "SOL/AVAX 크래시집중" 프레이밍을 slow-grind로 정정하도록 교정.
반증-우선 + 적대적 검증이 *내 자신의 분석*의 false-positive(틀린 EW 주장)를 잡은 사례.

## 캐비엇 (리포트 명시)
- **크래시 정의 한계**: BTC DD>30%가 전 히스토리의 **63%**(2023/3208일)를 "크래시"로 라벨
  (1015일·682일 침체 grind 포함). 실질은 "ATH 아래 vs 신고가" → 모든 분해 왜곡. 사전등록이라
  바꾸진 않았으나 SOL/AVAX 결과의 레버.
- **상관**: LUNA 윈도 자산 평균상관 0.76 → "5/6 포착"은 사실상 한 상관 이동. 유효 독립 N≈5.4/8
  → 교차자산 증거는 헤드카운트보다 약함(REJECTED 방향이라 판정엔 무해).
- **에피소드 thinning**: 실효 독립 에피소드 ~3개(2018베어는 2yr-train 탓 OOS 미커버 existed=0,
  FTX⊂2022베어) → 일반성은 "2/5"보다도 덜 검증됨.
- **생존편향**: 8자산=오늘 살아남은 코인 → 전 PnL 낙관 편향.
